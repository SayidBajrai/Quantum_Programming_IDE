Quick Start

Option 1: Code Editor

1. Click "Open Compiler" to start coding

2. Write your OpenQASM 3 code in the editor

3. View the real-time circuit diagram

4. Run simulations to see quantum state results

5. Save your circuits for future use

Option 2: Visual Circuit Builder

1. Click "Circuit Builder" to open the visual editor

2. Drag gates from the palette onto the circuit canvas

3. Add or remove qubits as needed

4. Watch QASM code generate in real-time

5. Run simulations or save your circuit
